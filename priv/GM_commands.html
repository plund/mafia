Mafia GM Commands

GM Command Effect on the Mafia tracker bot
PLAYER ... DIED
PLAYER ... DEAD
PLAYER ... BEEN LYNCHED
 The bot considers a PLAYER as DEAD when it sees a correctly spelled player name on one line in a GM message and one of "DIED", "DEAD" or "BEEN LYNCHED" later on the same line. The 3 "..." means that there may be other words in this position.

Method
1. Bot search for any of "DIED", "DEAD" or "BEEN LYNCHED" in GM messages.
2. Bot tries to find a correctly spelled player name before on the same line and sentence (case does not matter).

DAY ... HAS BEGUN
NIGHT ... HAS BEGUN
DAY ... HAS ENDED
NIGHT ... HAS ENDED 		The bot ends the current phase by moving the deadline of the current phase to the time of the GM message containing this text. The deadline times for later phases are not changed

Method
1. Bot search for "HAS ENDED" and for "HAS BEGUN" in GM messages.
2. Bot verfies that "DAY" or "NIGHT" can be found earlier on the line. (but it ignores if this text does not match the current phase type)

DEADLINE ... MOVED 24H LATER
DEADLINE ... MOVED 24H EARLIER 		The bot moves the deadline time of the current phase earlier or later with the number of hours given.

Method
1. Bot search for "DEADLINE" in GM messages.
2. Bot search for "LATER" or "EARLIER" after on same line
3. Before on the same line the bot will look for the number of hours

GAME HAS ENDED The bot put the game into phase end of game

Notes:

- Upper or lower case does not matter. The bots converts all text to upper case before it starts looking for GM commands.

- Before each game the bot is told which player ids the GMs have and which thread that is the game thread. The bot will look for GM commands in everything written by any GM in the game thread.
