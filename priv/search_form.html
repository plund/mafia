Mafia Search

Search in M25

User Names Zero, one or more correctly spelled user names
Contained words Zero, one or more words, separated either by space or "|".
Day/Night numbers examples: "3" or "2-5"


Quotes must be used to search for user names containing a space. Normally spaces divide up words so then you may need to use quotes ("") For example to search for "Hellenic Riot" quotes are needed.

Minimum one search condition must be specified. Either a one or more user names or one or more words that must be found in the message or one single day number.

Spaces (and) between words in the word condition, means that all words must be found in the message. If "|" (or) is used between the words only one of the words must be found in the message.

Spaces in the User name fields is "or". Do not use "|" here.

Upper or lower case does not matter in user name and word searches.
