Mafia Search

Search for Message in Current Game

User Names One or more user names or parts of user names.
Words in Message Zero, one or more words, separated either by space or "|".
Page(s) or Phase(s) Examples: day1, page 44, p11-17, D3, N3, n2 - d3, d3-, -p7 day 2 - night 2


Notes:

The MAX output of any search is 400k byte! You need to modify your search conditions if you reach this limit.

Upper or lower case does not matter in any type of search condition.

Quotes must be used to search for something containing a space. Normally spaces divide up words so then you may need to use quotes ("") For example to search for the full string "Hellenic Riot", quotes are needed.

Minimum one search condition must be specified. Either a one or more user names or one or more words that must be found in the message or one single day number.

In the User conditions, a space between two user means "or". Messages for both users are listed. No word boundary requirements apply.

In the Word conditions, a space between two words means "and". Both words need to exist in the message for it to match. Word boundary requirements DO apply.

The "|" character between two words means "or". Only one of the two words must be found in the message to match.

Word searches are normally done with word boundaries. Searching for "nd" matches the word "nd" but not the word "and".
Add the "*" character at the beginning or end to remove the word boundary requirement. Searching for "*nd*" matches "nd" but also "and" and "mandelbrot".
Doing a word search for "Jamie*|pete*" will result in all messages where any word starts with either "Jamie" or "pete"

Words searched for will be highlightened where they are found in the resulting output.

In the Page/Phase condition a single number is a page number. A page can also be given using "p33" or "page 33" Here you can also enter "d3" or "D3" or "day 3" which all will output messages for day 3. Same for "n1", "N 1" and "night 1".

You can give a sequence like "D3-N3", "page 1 - page 13" or even mixed "p30-d1". You may also start or end with dash. "-6" will result in pages 1-6, while "N4 -" returns messages for night 4 and everying coming after even End of Game messages.
