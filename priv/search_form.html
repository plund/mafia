Mafia Search

Search for Message in Current Game

User Names One or more user names or parts of user names.
Contained words Zero, one or more words, separated either by space or "|".
Day/Night numbers examples: "1" or "2-4", "end" or "4-end"


Notes:

Upper or lower case does not matter in any type of search condition. Case do not matter for the word "end" either.

Quotes must be used to search for something containing a space. Normally spaces divide up words so then you may need to use quotes ("") For example to search for the full string "Hellenic Riot", quotes are needed.

Minimum one search condition must be specified. Either a one or more user names or one or more words that must be found in the message or one single day number.

In the User condition a space between two user is "or". Messages for both users are listed.

In word conditions, a space between two words is "and", meaning that both words need to exist in the message for it to match. The "|" character between two words is or, meaning that only one of the two words must be found in the message. Words normally need to have word bondaries, so that a search for "nd" does not match the word "and". Add the "*" char at the beginning or end to remove the word boundary requirement. Searching for "*nd*" matches "and" but also "mandelbrot". Searching for "Jamie*|pete*" will result in all message where a word starts with either "Jamie" or "pete"

Words searched for will be highlightened in the resulting output.

Inserting anything starting with "end" in the day number field will display messages posted after the game has ended. It is also possible to search day number to end, e.g. "5-end".
